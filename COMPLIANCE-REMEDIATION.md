### 2.3 微信品牌名直接使用 — 商标侵权风险

**分级处理**：

| 场景 | 处理方式 |
|---|---|
| 项目标题 `# 微信助手` | → `摘星 — 本地消息管理工具` |
| 页面 `<title>` | → `摘星 · 本地助手` |
| 侧边栏标题 `微信助手` | → `摘星` |
| 欢迎页 `欢迎来到微信助手` | → `欢迎使用摘星` |
| 功能标签 `微信推送` | → `消息推送` |
| 功能标签 `微信收藏` | → `收藏管理` |
| 描述 `你的本地微信数据中心` | → `你的本地消息管理中心` |
| README `微信是我们每天使用最久的 App` | → `即时通讯是我们每天使用最久的工具` |
| 日志 `console.log('微信助手')` | → `console.log('摘星')` |

**原则**：**产品名不用"微信"二字，功能描述中"微信"替换为"即时通讯"或省略**。仅在"与微信客户端协同工作"这种技术说明中可保留，但加注释说明是"与用户已登录的客户端配合"。

|---|---|---|
